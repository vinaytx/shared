Nanobot + Ollama setup
=======================

Python Setup
============
1 - Setup pyenv for Windows
     https://github.com/pyenv-win/pyenv-win
2 - Install Python under pyenv
	
     -- pyenv install 3.12.10

     -- pyenv local  3.12.10
     -- pyenv global 3.12.10

     -- pyenv version   << check to make sure that you are using 3.12.10

Download Nanobot
================
   -- https://github.com/HKUDS/nanobot
   -- I have a modified version of Nanobot...  (https://github.com/vinaytx/shared.git)

Setup Nanobot
=============
   go into nanobot directory
    -- pip install -e .


Download markitdown   to convert files to text
===================
   -- https://github.com/microsoft/markitdown
   -- I have a modified version of markitdown with the correct version of mcp server SW. (markitdown.zip)
   
>> cd markitdown\packages

go in each directory
   >>  pip install -e .

Download Ollama
================
    --- https://www.ollama.com/

Setup Ollama account to use Ollama cloud
====================

Download Gemma 
==============
     --  ollama run gemma4:e2b
     --  ollama run gemma4:e4b

Download Gemma cloud
=====================
     -- ollama run gemma4:31b-cloud


Start Nanobot

>> nanobot onboard

Copy configuration file (config.json) from nanobot source directory to .\nanobot

>> cd <user dir>\.nanobot
## Steps to use local ollama model
>> copy C:\Shared\dev\nanobot\config.json  config.json   . 
## Steps to use Cloud ollama model
>> copy C:\Shared\dev\nanobot\config-ollama-cloud.json config.json

Start the MCP gateway

Window 1 >>> markitdown-mcp --http --host 127.0.0.1 --port 3001

Start nanobot ..  you will need to open 3 command terminals

Window 2 >>>  nanobot gateway

window 3 >>> nanobot agent

window 4 >>>  cd <user id>\.nanobot\logs
    Directory where debug logs that show messages going to the LLM
